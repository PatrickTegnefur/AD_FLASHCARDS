Hjälp & instruktioner

🃏 Spela med korten

  • Klicka på ett kort för att vända det och se svaret.
  • Betygsätt dig själv med Säker / Ganska säker / Osäker.
  • Använd filterpillsen (Kan, Halvklar, Kan ej, Mix) för att öva på specifika grupper.
  • Mix visar alla kort du inte är helt säker på — bra för repetition.
  • Aktivera 🔀 Slumpa för att blanda kortordningen.
  • Aktivera ↩️ Omvänd för att se svaret först och gissa frågan.

🏷️ Kategorier

  • Varje kort kan tillhöra flera kategorier (t.ex. "Vinklar" och "Geometri").
  • Skriv ett kategorinamn i kortets kategori-fält och tryck Enter — autocomplete föreslår befintliga kategorier.
  • Kategorierna visas som filter ovanför korten när minst en kategori finns.
  • Under 🏷️ Kategorier i editorn kan du byta namn på en kategori — det uppdateras på alla kort automatiskt.

✏️ Skapa och redigera kort

  • Klicka på ✏️ Kort i menyn för att lägga till nya kort eller redigera befintliga.
  • Klicka på pennan (✏️) bredvid ett kort i listan för att redigera det.
  • Varje kort kan ha en valfri bild — på framsidan, baksidan eller båda.

📥 Importera kort från Excel

  • Klicka på 📥 Importera i menyn.
  • Klistra in: Markera celler i Excel (A = Fråga, B = Svar, C = Kategori) → Ctrl+C → klistra in i fältet.
  • Ladda upp fil: Spara Excel-filen som .csv eller .xlsx och ladda upp direkt.
  • Kolumn C (kategori) är valfri — flera kategorier separeras med semikolon, t.ex. Vinklar;Geometri.
Tips: Kontrollera att kategorinamnen i Excel stavas likadant på alla kort — autocomplete i editorn hjälper dig att hålla det konsekvent.

💾 Dela kortlekar med elever

  • Gå till ⚙️ Inställningar och välj Spara kortlek (.json).
  • Dela .json-filen med elever via e-post, Teams, Google Classroom eller liknande.
  • Eleven öppnar appen, går till ⚙️ och väljer Öppna kortlek — klart! Inga konton behövs.
  • Bilder följer med automatiskt i .json-filen.
  • Exportera som .csv om du vill öppna korten i Excel igen.

🌙 Övrigt

  • Framsteg sparas automatiskt i webbläsaren — inga konton behövs.
  • Varje elev har sina egna framsteg på sin egen enhet.
  • Klicka på kortlekens namn i headern för att döpa om den.
  • Mörkt/ljust tema växlas med 🌙/☀️-knappen.
